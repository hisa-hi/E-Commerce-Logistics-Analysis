ブラジルEコマース：配送遅延が顧客満足度に与える影響の分析
1. プロジェクト概要
ブラジルの大手ECサイト「Olist」の公開データ（約10万件）を使用し、「物流の品質（配送遅延）」が「顧客の満足度（レビュー点数）」にどのような相関があるかを分析しました。

2. ビジネス上の課題
ECビジネスにおいて、配送の遅れは避けられないリスクですが、それが具体的にどれほどブランド価値（満足度）に影響し得るのかを定量的に把握し、物流改善への投資判断材料とすることを目的としています。

3. 分析プロセス
データ結合: 注文データ（orders）とレビューデータ（reviews）を結合。

データクレンジング:

日付文字列をdatetime型に変換。

配送済みデータのみを抽出し、欠損値を処理。

特徴量生成:

「予定到着日」と「実際の到着日」の差分から「遅延日数」を算出。

遅延の有無（0:なし、1:あり）のフラグを作成。

可視化と考察: Seaborn/Matplotlibを用いた相関分析。

4. 分析結果（インサイト）
遅延によるスコアの激減:

予定通りに届いた際の平均スコアは 約4.2点 ですが、1日でも遅延すると 約2.2点 まで下落します。

遅延が発生した時点で、満足度が約50%低下するという極めて強い負の相関が確認されました。

遅延日数との関係:

遅延が1〜5日増える間が最もスコアの下落が激しく、それ以降は低止まりする傾向にあります。

「少しの遅れ」が最も顧客の期待を裏切る要因となっていることがわかります。

5. 提言
物流網の最適化により「遅延1日以内」に抑える体制を構築することで、レビュー平均を大幅に改善できる可能性があります。

遅延が避けられない場合、早期のアラート通知やクーポン発行などのリカバリー策を「遅延確定直後」に行うことが重要です。
